Getting started

In this tutorial, we will show how to use this web site to get the most out of it.

Every tutorial usually consists of a number of pages, with highlighted Macaulay2 code for you to click or tap.

In each tutorial, click or tap on the highlighted Macaulay2 code. The code will be sent to the server, and evaluated. The result will be displayed in the right hand window. For example

R = QQ[x,y,z]
F = (x-1)^3 - (y-1)^3
factor F

To see help on a Macaulay2 topic,

To get started, select a tutorial from Index. Click on any highlighted code, Macaulay2 will execute it. The result is displayed on the right. Alternatively, you can use Input on the left to write your own commands. Execute a line or selection by positioning your cursor on it and click on the Evaluate button (or type Shift-Enter). You can switch back to the tutorial at any time.
